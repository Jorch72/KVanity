# README

### What is KVanity?
KVanity can get you custom addresses for [Krist](http://www.computercraft.info/forums2/index.php?/topic/22201-krist-minable-currency-that-works-across-servers/).

### How do I use KVanity?
1. Download the jar from the releases page
2. Open a terminal and change to the directory that contains the jar
3. `java -jar kvanity.jar -p kPREFIX -s SUFFIX`
4. Wait a while, addresses with long prefixes or suffixes will take a while to generate
5. Note that prefixes **must** begin with `k`.

### Options
    usage: java -jar kvanity.jar
     -p <arg>   Krist address prefix
     -s <arg>   Krist address suffix
     -v         Verbose
